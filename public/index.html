Pangea Wrapper

A wrapper for some of Pangea's services. Try:
  • /embargo?countryCode=CD to check the Democratic Republic of the Congo's embargo status, or try a different country code.
  • /isVpn?ip=2.56.189.74 to check whether 2.56.189.74 originates from a VPN, or try a different IPv4 address. No query parameter should check your IP. Note that it only works for IPv4 addresses.
See pangea.cloud for more information.
